To enable synchronization, we must have an agreement about how the data is going to be tranferred.

If you decide to accept, synchronizing your data across devices/browsers, it entales that; the server of this app, acts as the mediator to have the latest copy of your notes on each devices, in order to merge and serve to each other browser that you choose.
